# 安全说明

## 凭据与浏览器存储

- 不要提交 `.env`、导出的 SIP 信令、抓包文件或包含真实部署信息的日志。
- 网页为了方便切换话机，会将保存的 SIP 服务器地址、账号和密码写入当前浏览器的 `localStorage`。
- 因此不要在共享电脑或不可信电脑上保存话机配置；测试完成后应清除浏览器站点数据。

## 日志与信令

- 应用信令面板会隐藏 `Authorization` 与 `Proxy-Authorization` 头。
- 信令、日志和 SDP 仍可能包含号码、IP 地址、Call-ID、服务器信息或媒体地址。公开前必须脱敏。
- `webrtc-sip.log` 可能含有运行错误和网络端点，提交 Issue 前请人工检查。

## 漏洞反馈

不要在公开 Issue 中提交密码、私有服务器地址、可用账号或未脱敏抓包。请私下联系维护者并提供最小复现步骤和脱敏日志。
